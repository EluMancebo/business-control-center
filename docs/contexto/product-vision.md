# Product Vision

El objetivo es crear un sistema operativo digital
para pequeñas empresas locales.

## Fase A

20–30 clientes locales rentables.

Sistema interno optimizado para producir webs,
captación y automatización con eficiencia.

## Fase B

Escalar el sistema hacia un SaaS más autónomo
con panel más completo.    

## Usuario principal del panel

El usuario principal será el dueño del negocio, que usará el panel muchas veces desde móvil.  
Esto condiciona diseño, densidad, acciones y jerarquía de todas las interfaces.
