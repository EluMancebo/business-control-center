# Technical Directives

## Stack

- Next.js 15
- React 19
- Tailwind CSS v4
- TypeScript
- MongoDB

## Reglas técnicas

- No usar inline styles
- No usar any
- Cambios mínimos siempre
- Código completo cuando se modifica
- Validar con npm run lint
- Validar con npm run build
